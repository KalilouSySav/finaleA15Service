Documentation de l'API

Insertion de Données de Projet

Endpoint : /insert

Méthode : POST

Description : Cet endpoint est utilisé pour insérer des données sur un projet.

Paramètres :

  • code_projet (chaîne) - Le code du projet.
  • description (chaîne) - La description du projet.

Exemple de Requête :

POST /insert HTTP/1.1
Host:  http://127.0.0.1:8000/
Content-Type: application/x-www-form-urlencoded

code_projet=projet123&description=Description%20du%20projet

Récupération de Données de Projet

Endpoint : /get/{project_id}

Méthode : GET

Description : Cet endpoint est utilisé pour récupérer des données de projet en fonction de l'ID du projet.

Exemple de Requête :

GET /get_project_data/123 HTTP/1.1
Host:  http://127.0.0.1:8000/
